4. Confidence Interval & Descriptive Statistics

Metric Value Formula / Explanation Result
Mean 70 Mean = Σx / n 70
Standard Deviation (SD) 10 SD = sqrt[Σ(x - mean)² / n] 10
Sample Size (n) 100 n = number of observations 100
Confidence Interval (95%) CI = Mean ± Z × (SD / √n) Z = 1.96 for 95% 70 ± 1.96 × (10 / √100) = [68.04, 71.96]

Conclusion: The 95% Confidence Interval is [68, 72], meaning we are 95% confident the true population mean lies in this range.

5. Z-Test & T-Test for Hypothesis Testing

Test Formula Calculation Result Conclusion
Z-Test (Sample Mean vs Population Mean = 68) Z = (x̄ - μ) / (σ / √n) Z = (70 - 68) / (10 / √100) = 2.0 p ≈ 0.045 p < 0.05 ⇒ Reject H₀
T-Test (Group A vs Group B) t = (x̄₁ - x̄₂) / √[(s₁²/n₁)+(s₂²/n₂)] Example: x̄₁ = 72, x̄₂ = 68, t = 2.1 p = 0.04 p < 0.05 ⇒ Statistically Significant

Interpretation: The Z-Test and T-Test both show significant results (p < 0.05), so the null hypothesis (no difference) is rejected in both cases.

6. Normal, Binomial & Poisson Distributions

🧮 How These Were Calculated:

DistributionFormulaDescription
Normal (μ = 5, σ = 1.5) f(x) = (1/√2πσ) * e^(-(x−μ)²/2σ²) Bell-shaped curve; symmetric around mean. Approximated manually as [0, 0.1, 0.3, ..., 0]
Binomial (n = 10, p = 0.5) P(k) = C(n, k) * p^k * (1−p)^(n−k) Probability of getting exactly k successes in n trials. Shown for k = 0 to 10.
Poisson (λ = 4) P(k) = (λ^k * e^−λ) / k! Probability of k events in fixed interval. Approximated for k = 0 to 10.

7. Correlation & Regression

StepFormulaValue
Data Points (x, y) - (1,2), (2,4), (3,5), (4,8)
Mean of x (x̄) Σx / n (1+2+3+4)/4 = 2.5
Mean of y (ȳ) Σy / n (2+4+5+8)/4 = 4.75
Slope (b) Σ[(x - x̄)(y - ȳ)] / Σ[(x - x̄)²] b = [(1−2.5)(2−4.75)+(2−2.5)(4−4.75)+(3−2.5)(5−4.75)+(4−2.5)(8−4.75)] / [(1−2.5)²+(2−2.5)²+(3−2.5)²+(4−2.5)²] = 2.2
Intercept (a) a = ȳ − b × x̄ 4.75 − 2.2 × 2.5 = -0.75
Regression Line y = a + bx y = -0.75 + 2.2x
